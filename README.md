# Harsha AI Lab 🧪

A personal sandbox for building AI-powered experiments.
Not a product. Not a portfolio piece. A lab — where things break, ideas get tested, and something interesting always comes out.

**Live →** [harsha-ai-lab.github.io]
**Portfolio →** [polaharsha.com](https://polaharshavardhan455.github.io/harsha-portfolio/)

---

## What This Is

Most AI projects are just API wrappers dressed up nicely.
This lab is different — the goal is to go deeper.

- Real tools people can actually use
- Simulations that make AI concepts visible
- Hardware talking to intelligence (ESP32, sensors, real data)
- Open-source models running locally, not just cloud APIs
- Experiments that teach me something new every time

Built in public. Broken often. Always improving.

## Tech Stack

**Frontend** — HTML / CSS / JS → React (as it grows)

**Backend** — Node.js + Express

**AI** — Claude API (primary), open-source models locally (planned)

**Database** — SQLite → PostgreSQL (as needed)

**Hardware** — ESP32, Raspberry Pi, Python

**Hosting** — GitHub Pages (frontend), Railway or Render (backend)
